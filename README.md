## Unity-2D-Roguelike

> ### 유니티 2D 로그라이크 게임 만들기 강좌

유니티 공식 튜토리얼 영상입니다.   
본 강좌에서는 기본 사용법 부터 판, 레벨, 움직임, 음향, 모바일 기기에서의 컨트롤까지 전반적인 내용을 학습합니다.
